Cómo obtener API Key de Gemini

Obtén tu API Key de Gemini

Sigue estos sencillos pasos para obtener tu API key gratuita de Google Gemini.

1 Accede a Google AI Studio

Ve a la página de API keys de Google AI Studio haciendo clic en el botón de abajo.

Abrir Google AI Studio Se abrirá en una nueva pestaña: https://aistudio.google.com/app/apikey 2 Inicia sesión con tu cuenta de Google

Si no has iniciado sesión, Google te pedirá que lo hagas con tu cuenta de Google.

Puedes usar cualquier cuenta de Google personal No necesitas una cuenta empresarial 3 Crea una nueva API Key

Una vez en la página, verás un botón para crear una nueva API key.

Haz clic en "Create API key" o "Crear API key" Selecciona un proyecto de Google Cloud o crea uno nuevo La API key se generará automáticamente 4 Copia tu API Key

Una vez generada, verás tu API key en pantalla.

Haz clic en el ícono de copiar para copiar la API key ¡Importante! Guarda tu API key de forma segura 5 Pega la API Key en SnapReceipt

Regresa a la página de Configuración de SnapReceipt y pega tu API key en el campo correspondiente.

Ve a la página de Configuración Pega tu API key en el campo "Gemini API Key" Haz clic en "Guardar Configuración" ¡Listo!

Ahora puedes empezar a capturar recibos y la IA de Gemini extraerá automáticamente la información.

 Información Importante Gratis: Gemini 1.5 Flash tiene un plan gratuito generoso Privacidad: Tu API key se guarda de forma segura y solo tú puedes verla Personal: Cada usuario necesita su propia API key
